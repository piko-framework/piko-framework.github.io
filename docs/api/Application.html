Properties

public $basePath : string

The absolute base path of the application.

public $bootstrap : array

List of modules that should be run during the application bootstrapping process. Each module may be specified with a module ID as specified via [[modules]].

During the bootstrapping process, each module will be instantiated. If the module class implements the bootstrap() method, this method will be also be called.

public $charset : string

The charset encoding used in the application.

public $config : array

The configuration loaded on application instantiation.

public $defaultLayout : string

The default layout name without file extension.

public $defaultLayoutPath : string

The default layout path. An alias could be used.

public $errorRoute : string

The Error route to display exceptions in a friendly way. If not set, Exceptions catched will be thrown and stop the script execution.

public $language : string

The language that is meant to be used for end users.
